# Build::Cloud

Tools for building resources in AWS

## Installation

Add this line to your application's Gemfile:

    gem 'build-cloud'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install build-cloud

## Usage

See the command line help for `build-cloud`.
